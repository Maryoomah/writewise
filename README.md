# writewise

It is a platform that is designed for students and teachers; it helps bridge the gap between teacher feedback and student revision by using AI as a revision coach rather than a writing tool. 
The goal is to help students avoid getting stuck on how to implement teacher feedback with AI as a guide rather than a rewriting tool.

-React
-TypeScript
-Tailwind CSS
